# 🔍 LogicLens

LogicLens is a full-stack web application designed to [Yahan apne project ka short description likho - jaise: "track and solve coding problems"].

## 🚀 Features
* **Problem Tracking:** Add, edit, and list coding problems.
* **Full-Stack Architecture:** Separate frontend (`client`) and backend (`server`).
* **Secure Configuration:** Environment variables managed locally.

---

## 🛠️ Tech Stack
* **Frontend:** React.js, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (or your database)

---

## 💻 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites
* Node.js installed on your system.

cd backend/server
npm install

PORT=5000
MONGO_URI=your_mongodb_connection_string

Running the app
cd backend/server
npm start

cd client(frontend
npm start
